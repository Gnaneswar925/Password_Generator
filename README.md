# 🔐 Password Generator

A simple and responsive Password Generator built using JavaScript that creates strong, random passwords with a single click.

## Features

* Generates secure random passwords
* Includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Special symbols
* One-click password generation
* Copy password directly to clipboard
* Lightweight and beginner-friendly project

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. The generator ensures at least one uppercase letter, one lowercase letter, one number, and one special character.
2. Remaining characters are filled randomly from all available character sets.
3. A 12-character strong password is generated and displayed.
4. Users can copy the generated password with a single click.
